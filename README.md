# Телеграм бот для работы с панелью 3x-ui

## Функционал:
- Создание нового клиента
- Подтверждение верификации клиента со стороны админа
- Напоминание об оплате клиентам
- todo: Управление клиентами (удаление, блокировка, разблокировка)

## Требуемые библиотеки:
`pip install python-dotenv`
`pip install aiogram`
`pip install apscheduler`
